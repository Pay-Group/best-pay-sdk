#best pay sdk -- 可能是目前最好的支付sdk
## 支付方式支持
1. 支付宝（app, pc, 扫码等）
2. 微信（app, h5, 扫码等）

## 功能支持
1. 支付
2. 退款（自动秒退）

## 特别注意
1. 本sdk要求的最低的jdk版本是8(纳尼, 你还在用7? 难道你不知道9快出来了吗)
2. 支付时的编码只支持utf-8
3. 如果想贡献代码，请阅读[代码贡献指南](https://github.com/Pay-Group/best-pay-sdk/blob/master/CONTRIBUTION.md)
